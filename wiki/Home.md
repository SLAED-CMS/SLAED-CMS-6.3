## üìä System Requirements

### Minimum Requirements
- **PHP:** 8.0+
- **Database:** MySQL 5.7+ / MariaDB 10.3+
- **Web Server:** Apache 2.4+ / Nginx 1.14+
- **Memory:** 128MB RAM
- **Disk Space:** 50MB

### Recommended Setup
- **PHP:** 8.1+ with OPcache
- **Database:** MySQL 8.0+ / MariaDB 10.6+
- **Memory:** 256MB+ RAM
- **Storage:** SSD drive
- **CDN:** For static assets
