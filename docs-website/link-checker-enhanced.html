Enhanced Link Checker

This tool checks all documentation links for broken references using server-side PHP.

How it works: This tool uses a PHP script to check if documentation files exist on the server. This method is more reliable than client-side JavaScript checking.

Troubleshooting

If links show as broken:

  • Verify the file exists in the documentation directory
  • Check that the filename matches exactly (case-sensitive)
  • Ensure there are no typos in the filename
  • Check file permissions (must be readable by the web server)
