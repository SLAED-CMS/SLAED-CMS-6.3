Erweiterter Link-Prüfer

Dieses Werkzeug prüft alle Dokumentationslinks auf fehlerhafte Verweise unter Verwendung von serverseitigem PHP.

So funktioniert es: Dieses Werkzeug verwendet ein PHP-Skript, um zu prüfen, ob Dokumentationsdateien auf dem Server existieren. Diese Methode ist zuverlässiger als clientseitige JavaScript-Prüfung.

Link-Status

Bereit zur Link-Prüfung. Klicken Sie auf die Schaltfläche oben, um zu starten.

Fehlerbehebung

Wenn Links als fehlerhaft angezeigt werden:

  • Überprüfen Sie, ob die Datei im Dokumentationsverzeichnis existiert
  • Stellen Sie sicher, dass der Dateiname genau übereinstimmt (Groß-/Kleinschreibung beachten)
  • Vergewissern Sie sich, dass keine Tippfehler im Dateinamen vorhanden sind
  • Prüfen Sie die Dateiberechtigungen (muss vom Webserver lesbar sein)
